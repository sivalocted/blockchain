- P2P networking

## Out of scope

- Physical attacks on infrastructure

## Severity
- Critical: chain halt, double spend, key compromise
- High: remote crash, consensus divergence
- Medium: denial of service with limited scope
- Low: UI or non-critical bugs

## Disclosure
